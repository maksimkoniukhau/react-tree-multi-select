# Controlled

**react-tree-multi-select** allows you to externally control the dropdown’s open and close state using the `openDropdown` prop.<br/>
This gives you full control over when the dropdown is rendered or hidden.<br/>
By pairing `openDropdown` with the `onDropdownToggle` callback, you can synchronize the dropdown’s visibility state with your application’s state,
enabling custom behaviors such as toggling from external controls or reacting to other UI events.<br/>
